
# Flashcard ChatGPT App

This is a flashcard ChatGPT app designed to help you study for your exams. It provides an interactive flashcard experience powered by ChatGPT, making learning engaging and effective.

## Demo

![App Usage GIF](./assets/demo.gif)

Try the app live: [study-buddy-aedacb7d.alpic.live/](https://study-buddy-aedacb7d.alpic.live/)

## Tools

- **createFlashcardDeck**: Configure your study preferences (language, difficulty, deck size)
- **startStudySession**: Study with interactive flashcards, flip animations, and progress tracking
